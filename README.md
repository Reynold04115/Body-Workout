## 📑 1. Project Abstract
The **Dynamic Fitness & Workout Recommendation System** is a frontend web application designed to provide personalized strength training routines based on user biometrics. Recognizing the lack of tailored guidance for beginners in fitness, this system utilizes a custom algorithmic approach to calculate safe, effective starting weights based on a user's current weight, height, and body morphotype (Ectomorph, Mesomorph, Endomorph). 

## 🎯 2. Problem Statement
Generic workout plans often lead to injury or lack of progress due to inappropriate starting weights and volume. This project aims to solve this by building an interactive, accessible, and responsive web interface that instantly calculates personalized training protocols without requiring a heavy backend infrastructure or user account creation.

## ⚙️ 3. System Modules & Features
* **Biometric Assessment Module:** Captures user inputs (Height, Weight, Body Type) and validates the data.
* **Algorithmic Weight Calculator:** Processes user data against a predefined ratio matrix to suggest optimized starting weights in both Kilograms (KG) and Pounds (LBS).
* **Dynamic DOM Filtering:** Utilizes JavaScript array filtering to sort exercises by specific muscle groups (Chest, Back, Legs, Core) seamlessly without page reloads.
* **Interactive UI/UX:** Implements modern web design principles including glassmorphism, responsive grid layouts, and visual feedback mechanisms (custom cursors, CSS animations).

## 💻 4. Technology Stack
This project was developed utilizing a lightweight, pure frontend architecture to ensure maximum speed and cross-browser compatibility.

* **Markup & Structure:** HTML5
* **Styling & Animation:** CSS3 (Custom Properties, Flexbox, CSS Grid)
* **Logic & DOM Manipulation:** Vanilla JavaScript (ES6+)
* **Version Control:** Git & GitHub

## 🚀 5. Local Execution (Setup Instructions)
Since this project relies on a serverless frontend architecture, installation is straightforward and requires no database configuration.
